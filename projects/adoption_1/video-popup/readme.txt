== Installation ==

### Installation

1. Upload 'video-popup' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now use this shortcode: [video_popup url="" text="" auto=""]

###Shortcode

    [video_popup url="" text="" auto=""]

    url=”here enter youtube or vimeo video link only”, default is none.
    text=”here enter your text, for example: my video“, default is same video link.
    To disable video autoplay use auto=”no”, default is “yes”.
